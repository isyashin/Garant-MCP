# Use Case: Договорная работа

## Описание

Составление и проверка договоров (подряда, оказания услуг, поставки, аренды, купли-продажи), претензии по договорам.

## Типовые запросы пользователя

- «Помоги составить договор подряда»
- «Проверь договор аренды»
- «Контрагент не оплатил, составь претензию»
- «Нужен договор оказания услуг»

## Какие нормы искать

- ГК РФ (общие положения, договор подряда, поставки, аренды, услуг)
- Постановление Пленума ВС РФ по договорным спорам
- ФЗ "О защите прав потребителей" (если сторона — потребитель)

## Уточняющие вопросы

1. Кто стороны (наименование, реквизиты)?
2. Вид договора (подряд, услуги, поставка, аренда, купля-продажа)?
3. Предмет договора (работы, товары, услуги)?
4. Стоимость, порядок и сроки оплаты?
5. Сроки исполнения, ответственность за просрочку?
6. Порядок приёмки работ/товаров?
7. Нужен ли досудебный порядок урегулирования споров?

## Какой документ готовить

- Договор.docx
- Соглашение.docx
- Претензия.docx
- Исковое_заявление.docx

## Сроки

- Исковая давность по договорным обязательствам: 3 года.
- Досудебный порядок: обычно 30 дней, если предусмотрен договором.

## Риски

- Неполные или односторонние условия.
- Отсутствие ответственности за нарушение.
- Непонятный порядок приёмки.
- Неправильный выбор правового режима.
