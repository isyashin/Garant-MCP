# Use Case: Семейные споры

## Описание

Развод, алименты, раздел имущества, определение места жительства детей, порядок общения, брачный договор.

## Типовые запросы пользователя

- «Как подать на алименты?»
- «Хочу развестись и разделить имущество»
- «Бывший муж не платит алименты»
- «Как оформить брачный договор?»

## Какие нормы искать

- Семейный кодекс РФ
- ГК РФ (раздел имущества, брачный договор)
- Постановление Пленума ВС РФ №15, №9, №10
- ФЗ "Об актов гражданского состояния"

## Уточняющие вопросы

1. Стороны (ФИО, даты рождения, адреса)?
2. Дата заключения/расторжения брака?
3. Дети (ФИО, даты рождения, с кем проживают)?
4. Имущество, подлежащее разделу?
5. Доходы сторон?
6. Необходим ли брачный договор?
7. Какой результат ожидается?

## Какой документ готовить

- Исковое_заявление.docx (о расторжении брака, алиментах, разделе)
- Соглашение.docx (об уплате алиментов, о разделе имущества)
- Договор.docx (брачный договор)

## Сроки

- Иск о расторжении брака: без ограничений.
- Алименты: до достижения ребёнком 18 лет.
- Исковая давность по разделу имущества: 3 года.

## Риски

- Неправильный расчёт алиментов.
- Скрытие доходов ответчиком.
- Споры о месте жительства детей.
- Оспаривание брачного договора.
