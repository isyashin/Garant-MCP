# Use Case: Защита прав потребителей

## Описание

Возврат/обмен товаров, некачественные услуги, просрочка поставки, нарушение сроков ремонта.

## Типовые запросы пользователя

- «Помоги составить претензию на возврат автомобиля»
- «Некачественный ремонт телефона, как вернуть деньги»
- «Продавец не возвращает деньги за товар»
- «Просрочили доставку, хочу неустойку»

## Какие нормы искать

- Закон РФ «О защите прав потребителей»
- ГК РФ (договор розничной купли-продажи, подряда, оказания услуг)
- Постановление Пленума ВС РФ №7 о применении ЗоЗПП
- Постановление Правительства РФ о перечне товаров, не подлежащих обмену

## Уточняющие вопросы

1. Кто вы (ФИО, паспортные данные, адрес)?
2. К кому обращаетесь (наименование продавца/исполнителя, адрес, реквизиты)?
3. Что приобреталось (товар/услуга/работа), дата и сумма?
4. Какие документы имеются (чек, договор, гарантийный талон, акт)?
5. В чём заключается нарушение (дефект, просрочка, отказ)?
6. Какой результат хотите (возврат денег, замена, уменьшение цены, неустойка)?
7. Когда и как обращались ранее (претензия, ответ)?

## Какой документ готовить

- Претензия.docx
- Исковое_заявление.docx (если претензия не удовлетворена)

## Сроки

- 10 дней на рассмотрение претензии по ЗоЗПП.
- Исковая давность: 3 года (ст. 196 ГК РФ).
- Для досудебного урегулирования по ряду споров обязательна претензия.

## Риски

- Неполный пакет доказательств.
- Просрочен срок подачи претензии/иска.
- Неправильный расчёт неустойки.
- Отсутствие доказательств направления претензии.

## Алгоритм работы агента

```
1. create_case_tool("Защита_прав_потребителей_...")
2. create_case_description_tool_wrapper(case_path, "Возврат денег за ...")
3. get_usage_limits_tool()
4. search_and_cite_tool("защита прав потребителей возврат денег")
5. search_judicial_practice_tool("возврат товара ненадлежащее качество")
6. save_to_case_tool_wrapper(...) в служебное для агента/
7. Уточняющие вопросы пользователю
8. find_template_by_request_tool_wrapper("претензия")
9. copy_template_to_case_tool_wrapper("Претензия.docx", case_path, "Претензия.docx")
10. render_docx_template_tool_wrapper(...)
```
