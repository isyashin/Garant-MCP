- [Защита прав потребителей](Защита_прав_потребителей.md)
- [Трудовые споры](Трудовые_споры.md)
- [Жилищные споры](Жилищные_споры.md)
- [Семейные споры](Семейные_споры.md)
- [Договорная работа](Договорная_работа.md)
- [Наследство](Наследство.md)
- [Земля и недвижимость](Земля_и_недвижимость.md)

## Общий алгоритм агента

Для любого сценария:

1. **Создать кейс** — `create_case_tool`
2. **Сохранить запрос** — `create_case_description_tool_wrapper`
3. **Проверить лимиты** — `get_usage_limits_tool`
4. **Исследовать право** — `search_and_cite_tool`, `search_judicial_practice_tool`
5. **Сохранить исследование** — `save_to_case_tool_wrapper`
6. **Задать уточняющие вопросы** — через диалог
7. **Подобрать шаблон** — `find_template_by_request_tool_wrapper`
8. **Скопировать шаблон в кейс** — `copy_template_to_case_tool_wrapper`
9. **Заполнить шаблон** — `render_docx_template_tool_wrapper`
10. **Проверить и выдать документ** — сохранить в `результат/`

## Экономия лимитов

- Используйте `search_and_cite_tool` для поиска норм.
- Экспорт (`export_document_*`) — только по запросу пользователя.
- Ссылки на документы: `https://internet.garant.ru/#/document/{topic}/entry/{entry}`
