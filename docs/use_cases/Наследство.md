# Use Case: Наследство

## Описание

Принятие наследства, оформление права собственности, споры между наследниками, завещание, оспаривание завещания.

## Типовые запросы пользователя

- «Как вступить в наследство?»
- «Помоги оспорить завещание»
- «Наследники не дают мне долю»
- «Как составить завещание?»

## Какие нормы искать

- ГК РФ (наследственное право)
- Постановление Пленума ВС РФ №9 о наследственных делах
- ФЗ "О нотариате"

## Уточняющие вопросы

1. Кто умер (ФИО, дата смерти, последнее место жительства)?
2. Есть ли завещание, когда и где составлено?
3. Кто наследники (степень родства, ФИО)?
4. Какое имущество входит в наследственную массу?
5. Обращались ли к нотариусу?
6. Есть ли споры между наследниками?
7. Какой результат ожидается?

## Какой документ готовить

- Исковое_заявление.docx
- Соглашение.docx (о разделе наследства)
- Правовое_заключение.docx

## Сроки

- Срок для принятия наследства: 6 месяцев.
- Исковая давность по наследственным спорам: 3 года.

## Риски

- Пропущен срок принятия наследства.
- Завещание оформлено с нарушениями.
- Неучтённые долги наследодателя.
- Споры о долях наследников.
