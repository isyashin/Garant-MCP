# Use Case: Земля и недвижимость

## Описание

Купля-продажа недвижимости, сервитут, межевание, сделки с землёй, признание права собственности, споры с застройщиком.

## Типовые запросы пользователя

- «Как составить договор купли-продажи квартиры?»
- «Сосед захватил часть участка, что делать?»
- «Застройщик сорвал сроки сдачи дома»
- «Помоги оформить сервитут»

## Какие нормы искать

- ГК РФ (право собственности, сервитут, общая долевая собственность)
- Земельный кодекс РФ
- Градостроительный кодекс РФ
- ФЗ "Об участии в долевом строительстве" (214-ФЗ)
- Постановление Пленума ВС РФ №10, №7

## Уточняющие вопросы

1. Какой объект (квартира, дом, земельный участок, нежилое помещение)?
2. Адрес, кадастровый номер?
3. Кто стороны (ФИО/наименование, реквизиты)?
4. Есть ли документы (выписка ЕГРН, договор, акт приёма-передачи)?
5. В чём проблема (сроки, качество, границы, право собственности)?
6. Какой результат ожидается?

## Какой документ готовить

- Договор.docx (КП недвижимости, аренда земли)
- Претензия.docx (застройщику)
- Исковое_заявление.docx
- Соглашение.docx (о границах, о сервитуте)

## Сроки

- Исковая давность по недвижимости: 3 года.
- Для долевого строительства: особые сроки по 214-ФЗ.

## Риски

- Отсутствие регистрации права собственности.
- Неправильные кадастровые границы.
- Обременения/аресты на объект.
- Сложности с доказыванием захвата земельного участка.
