## Сценарий: Возврат автомобиля

### Шаг 1: Создание кейса

```python
# Создаем папку для кейса
await create_case("Возврат_автомобиля_2024")
# Результат: results/cases/20240601_123456_Возврат_автомобиля_2024/
```

### Шаг 2: Исследование

```python
# Ищем законодательство
laws = await search_documents("защита прав потребителей автомобиль гарантийный ремонт")

# Изучаем судебную практику
practice = await search_judicial_practice("возврат автомобиля гарантийный ремонт 45 дней")

# Сохраняем результаты
await save_text_document(laws, "законодательство.txt", "documents")
await save_text_document(practice, "судебная_практика.txt", "practice")
```

### Шаг 3: Уточняющие вопросы

Агент задает пользователю:
1. Какие неисправности обнаружены?
2. Сколько дней автомобиль в ремонте?
3. Какие документы имеются (ДКП, гарантийный талон, акты приема-передачи)?
4. Какой результат ожидается (возврат денег, замена авто)?

## Примечания

- Все документы сохраняются локально
- Агент никогда не фантазирует - использует только реальные данные
- Перед созданием документов обязательно задаются уточняющие вопросы
- Все действия логируются
