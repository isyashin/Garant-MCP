# Агент Гарант - Юрист

## Описание

Интеллектуальный агент для подготовки юридических документов на основе системы Гарант.

## Возможности

- Поиск и анализ законодательства
- Изучение судебной практики
- Работа с образцами документов
- Подготовка юридических документов
- Управление кейсами

## Использование

### Запуск агента

```bash
# Запуск MCP сервера
python -m garant_mcp.server

# Или через opencode
opencode --agent garant-lawyer
```

### Рабочий процесс

1. **Исследование** - Агент изучает законодательство по теме
2. **Уточнение** - Задает уточняющие вопросы пользователю
3. **Подготовка** - Создает документы на основе данных
4. **Проверка** - Проверяет готовые документы

### Структура папок

```
results/
├── cases/          # Кейсы и примеры
├── templates/      # Шаблоны документов
├── documents/      # Найденные документы
├── practice/       # Судебная практика
├── client_data/    # Данные клиента
└── output/         # Готовые документы
```

## Новые инструменты

### Работа с файлами

- `save_text_document` - Сохранить документ
- `load_text_document` - Загрузить документ
- `list_saved_documents` - Список документов
- `create_case` - Создать папку кейса
- `copy_template_file` - Копировать шаблон

## Пример использования

```python
# Создать новый кейс
await create_case("Возврат автомобиля")

# Найти законодательство
results = await search_documents("защита прав потребителей автомобиль")

# Изучить практику
practice = await search_judicial_practice("возврат автомобиля гарантийный ремонт")

# Сохранить результаты
await save_text_document(results, "research.txt", "documents")
await save_text_document(practice, "practice.txt", "practice")

# Подготовить документ
await create_legal_document("Претензия на возврат автомобиля...")
```

## Безопасность

- Никогда не фантазирует и не придумывает нормы
- Использует только проверенные источники
- Задает уточняющие вопросы перед созданием документов
- Сохраняет промежуточные результаты

## Конфигурация

Настройки агента в `docs/agent-config.json`:

```json
{
  "name": "garant-lawyer-agent",
  "version": "1.0.0",
  "output_dir": "results/output",
  "temp_dir": "results/temp"
}
```

## Требования

- Python 3.11+
- Доступ к API Гарант
- Установленные зависимости из requirements.txt
